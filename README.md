## 🎨 Dual Layout Styles

### 1. Minimalist Elegant Portfolio (`index.html`)
*   **Design Concept**: Clean editorial layout inspired by high-end design magazines and modern tech websites.
*   **Aesthetics**: Neutral tones, high-contrast typography, structural glassmorphic navigation panels, and smooth layout grids.
*   **Typography**: Cormorant Garamond (refined serif) and Space Mono (clean developer typeface) via Google Fonts.

### 2. Synthwave Cyberpunk Portfolio (`cyberpunk-portfolio.html`)
*   **Design Concept**: Immersive retro-arcade interface utilizing neon indicators and virtual CRT terminal scanlines.
*   **Aesthetics**: Sleek dark base (`#0a0a0f`) decorated with glowing pink, cyan, and yellow borders and keyframe flickering micro-animations.
*   **Typography**: Orbitron and VT323 (8-bit style typography) via Google Fonts.

---

## 🛠️ Features

*   🌓 **Theme Modes**: Native dark/light mode toggles on both portfolio versions.
*   📜 **Smooth Scrolling Nav**: Direct anchor-link navigation with scroll-behavior mapping.
*   🏗️ **Interactive Timeline**: Clean chronological overview of experience, milestones, and education.
*   📂 **Project Gallery**: Card grid layouts for linking and showcasing individual repositories.

---

## 💻 Tech Stack

*   **Languages**: HTML5, CSS3, Vanilla JavaScript.
*   **Animations**: Custom CSS keyframes, hover translations, glowing text-shadow animations.
*   **Icons & Assets**: Custom inline SVG elements and vector shapes.

Explore both themes locally:
